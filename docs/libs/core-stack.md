# Stack Technique Core 🏗️

## Frontend

- **Vanilla JavaScript** (ES6+) : Pas de framework lourd (React/Vue) pour garantir une performance maximale et une légèreté de chargement.
- **MapLibre GL JS** : Gestion de la performance cartographique.
- **CSS3 (Variables & Glassmorphism)** : Interface moderne, sombre et semi-transparente.

## Outillage

- **Vite** : Utilisé pour le bundling ultra-rapide et le Hot Module Replacement (HMR) en développement.
- **Node.js** : Moteur d'exécution pour les outils de build.

## Déploiement

Le projet est un site statique (SPA). Il peut être hébergé sur n'importe quel service de fichiers statiques (Vercel, GitHub Pages, Netlify, AlwaysData) après avoir généré le dossier `/dist`.
